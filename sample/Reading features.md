# Reading features

You arrived here by clicking an in-app link — ZIMD resolved the relative path and
opened this file in the reader, keeping you in flow.

## Try these

- Press **⌘F** and search this page; use **Enter** / **Shift+Enter** to cycle matches.
- Hover any code block and use the **Copy** button:

  ```bash
  echo "copied straight from ZIMD"
  ```

- Scroll down, open another file, then come back — your **reading position** is
  remembered per file.
- Check the title bar for this document's **word count and reading time**.
